# ⚽ FC26 World Draft (Fan Game)

재미로 만들어본 **해외 축구 드래프트 미니 게임**입니다.
자신만의 드림 팀 스쿼드를 짜고 공유해보세요!

## 🎮 게임 소개
좋아하는 선수들로 최고의 11인 스쿼드를 완성하는 웹 게임입니다.
운과 실력으로 최고의 케미스트리를 만들어보세요.

### 주요 기능
*   **5대 리그 / 전체 선수 데이터 지원**: 검증된 주요 리그 선수부터 전 세계 모든 선수까지 폭넚게 선택 가능.
*   **랜덤 드래프트 시스템**: 각 포지션마다 3명의 랜덤 후보 중 최적의 선수를 선택하세요.
*   **특별한 보정 시스템**: 팀 구성이 어려울 때를 대비해, 최소한의 스쿼드가 완성되도록 돕는 **숨겨진 보정 규칙**들이 존재합니다. (초반에 망했다고 포기하지 마세요!)
*   **케미스트리 확인**: 같은 팀, 국적, 리그 선수를 모으면 추가 정보를 보여줍니다.

---

## 🚀 플레이 방법
1.  **[게임 링크 접속](https://dodgood89-debug.github.io/draw_fc26_squad/)** (배포 후 주소 변경 필요)
2.  빈 포지션 카드를 클릭하여 후보 선수 3명 중 1명을 선택합니다.
3.  11명을 모두 채우고 나만의 스쿼드를 완성하세요.
